===============
Getting Started
===============

The ``kohonen`` package is a set vector quantizers in the style of the Kohonen
Self-Organizing Map.

.. _qs-setup:

Installation
============

It's easiest to install the ``kohonen`` package using ``pip``::

  pip install kohonen

The command will automatically install all dependencies.
